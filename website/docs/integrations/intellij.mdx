<a href="https://plugins.jetbrains.com/plugin/26096-configcat-feature-flags" target="_blank">ConfigCat's JetBrains/IntelliJ IDE plugin</a> to manage feature flags from IDEs.

Connect your ConfigCat Product and Config to your JetBrains/IntelliJ IDE. Find your Feature Flag's usages in your code easily.

This guide will help you install the JetBrains/IntelliJ IDE Plugin and familiarize you with its usage.

## Feature overview

- View your Feature Flags within the IDE.
- Create Feature Flags within the IDE.
- Open a Feature Flag on the ConfigCat Dashboard.
- Find Feature Flag usages in your code.
- Copy a Feature Flag's key to the clipboard.
- View your Products & Configs.
- Create Configs within the IDE.
- Open a Config on ConfigCat Dashboard.

## Install plugin

### Install within the IDE from Marketplace

<Steps>
<Step>
In your IDE, open the **File** > **Settings/Preferences** menu.
</Step>
<Step>
Go to the **Plugin** page and select the **Marketplace** tab.
</Step>
<Step>
Search for **ConfigCat Feature Flags**.
</Step>
<Step>
**Install** the plugin.
</Step>
<Step>
Configure the plugin ([see below](#configure-extension)).
</Step>
</Steps>

### Install manually within the IDE

<Steps>
<Step>
Download the latest release from [GitHub](https://github.com/configcat/intellij-plugin/releases/latest) or from [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/26096-configcat-feature-flags).
</Step>
<Step>
In your IDE, open the **File** > **Settings/Preferences** menu.
</Step>
<Step>
Go to the **Plugin** page and open the **Gear Icon** menu.
</Step>
<Step>
Select the **Install plugin from disk...** option.
</Step>
<Step>
Configure the plugin ([see below](#configure-extension)).
</Step>
</Steps>

### Advanced

This section is for you if you use a dedicated hosted/[on-premise](https://configcat.com/on-premise/) ConfigCat instance. In that case, you can specify your custom ConfigCat URLs  in the IDE.

<Steps>
<Step>
In your IDE, open the **File** > **Settings/Preferences** menu.
</Step>
<Step>
Search for **ConfigCat Feature Flags** page.
</Step>
<Step>
Fill out the **Public Api Base URL** and **Dashboard Base URL** inputs.

- `Public Api Base URL`: the base URL for the ConfigCat Public Management API. Defaults to: https://api.configcat.com.
- `Dashboard Base URL`: the base URL for ConfigCat Dashboard. Defaults to: https://app.configcat.com.
</Step>
</Steps>

## Usage of ConfigCat Feature Flags Plugin

The ConfigCat Feature Flags Tool Window can be opened by clicking the ConfigCat Feature Flags icon on the side of the IDE main window.

### Feature Flags & Settings tab

After you successfully configured your ConfigCat account in the **Settings**, you can use the **Feature Flags & Settings** tab and:

- View all of your Products, Configs and Feature Flags.
- Create new Configs and Feature Flags.
- Open your Configs and Feature Flags on the ConfigCat Dashboard.
- Copy a Feature Flag's key to the clipboard.
- Find your Feature Flag's usages in your code.

  <source src="/docs/assets/intellij/usage2.mp4" type="video/mp4"/>

</video>

#### Create a new Feature Flag
<Steps>
<Step>
Select the Config in the tree where you want to create your new flag.
</Step>
<Step>
Click the **+** button on the action bar to open the **Create Flag** dialog.
</Step>
<Step>
Select the setting type.
</Step>
<Step>
Enter the new Feature Flag name.
</Step>
<Step>
Enter the new Feature Flag key.
</Step>
<Step>
Add a hint to your Feature Flag (optional).
</Step>
<Step>
Select **OK** to confirm the Flag creation.
</Step>
</Steps>

#### Copy a Feature Flag's key to the clipboard
<Steps>
<Step>
Select the Feature Flag in the tree whose key you want to copy.
</Step>
<Step>
Click the **Copy** button on the action bar to copy the key to the clipboard.
</Step>
</Steps>

#### Open your Feature Flags on the ConfigCat Dashboard
<Steps>
<Step>
In the tree, select the Feature Flag you want to open in the Dashboard.
</Step>
<Step>
Click the **Open Config in ConfigCat Dashboard** button on the action bar to open the Feature Flag in the Dashboard.
</Step>
</Steps>

#### Find your Feature Flag's usages in your code
<Steps>
<Step>
Select the Feature Flag in the tree you want to search.
</Step>
<Step>
Click the **Find Usage** button on the action bar.
</Step>
<Step>
See the search results in the opened dialog.
</Step>
</Steps>

#### Create a new Config under a product
<Steps>
<Step>
Select the Products in the tree where you want to create your new config.
</Step>
<Step>
Click the **+** button on the action bar to open the **Create Config** dialog.
</Step>
<Step>
Enter a name for the new config.
</Step>
<Step>
Add a hint to your config (optional).
</Step>
<Step>
Select **OK** to confirm the Config creation.
</Step>
</Steps>

#### Open your Configs on the ConfigCat Dashboard

<Steps>
<Step>
Select the Config which you want to open in the Dashboard.
</Step>
<Step>
Click the **Open Config in ConfigCat Dashboard** button on the action bar to open the Config in the Dashboard.

:::info
The actions are available by right click menu as well.
:::
</Step>
</Steps>
