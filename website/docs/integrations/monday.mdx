The <a href="https://monday.com/marketplace/10000079" target="_blank">ConfigCat Feature Flags monday app</a> allows you to connect your Monday items and feature flags. Create or link existing flags to your items without leaving your monday instance.

Turn features On/Off right from a linked item on your Monday board. You can also easily modify the linked flags to edit or add new Targeting or Percentage Rules.

This guide will help you with the plugin installation and familiarise you with the plugin usage.

## Installation

<Steps>
<Step>
Install <a href="https://monday.com/marketplace/10000079" target="_blank">ConfigCat Feature Flags</a> monday.com app to your monday account.
</Step>
<Step>
Open one of your boards on monday.com and click on an Item.
</Step>
<Step>
Add the ConfigCat Feature Flags monday app to your Item.
</Step>
<Step>
To use ConfigCat Feature Flags, you must first authorize it with your ConfigCat Public API credentials.
</Step>
<Step>
Copy your ConfigCat Public API credentials to the inputs. Read more about <a href="https://app.configcat.com/my-account/public-api-credentials" target="_blank">ConfigCat Public API credentials here</a>.
</Step>
<Step>
Click **Authorize**.

:::info
Every monday.com user must authorize ConfigCat in monday.com who wants to use the ConfigCat Feature Flags app.
:::

  Your browser does not support the video tag.
</video>
</Step>
</Steps>


## Usage

### Linking existing feature flags

<Steps>
<Step>
Open any Item on your monday.com board.
</Step>
<Step>
Push the **Link existing feature flag** button.
</Step>
<Step>
Select a Product, Config, Environment, and Feature Flag to be linked to your monday.com Item.
</Step>
<Step>
When linked, you can manage the selected feature flag from this monday.com Item.

<video className="video" controls muted>
  <source src="/docs/assets/monday/link_existing_ff.mp4" type="video/mp4"/>
  Your browser does not support the video tag.
</video>
</Step>
</Steps>


### Creating new feature flags

<Steps>
<Step>
Open any Item on your monday.com board.
</Step>
<Step>
Push the **Create and link feature flag** button.
</Step>
<Step>
Select a Product and Config where you want to create the feature flag.
</Step>
<Step>
Set up your feature flag.
</Step>
<Step>
Select which environment you would like to link to this Item.
</Step>
<Step>
When linked, you can manage the selected feature flag from this monday.com Item.

<video className="video" controls muted>
  <source src="/docs/assets/monday/create_ff.mp4" type="video/mp4"/>
  Your browser does not support the video tag.
</video>
</Step>
</Steps>

### View and Edit linked feature flags

<Steps>
<Step>
Open any Item on your monday.com board.
</Step>
<Step>
See the linked feature flags in the **Feature Flag (ConfigCat)** view.
</Step>
<Step>
You can manage the selected feature flag from this Item.
</Step>
<Step>
You can add new Targeting Rules or Percentage Options to Feature Flags.
</Step>
<Step>
You can add new Targeting Rules with User, Segment or Prerequisite conditions with a large selection of Comparators.  [Read more about Targeting.](../targeting/targeting-overview.mdx)
</Step>
<Step>
You can remove Targeting Rules as well.

  <source src="/docs/assets/monday/edit_ff.mp4" type="video/mp4"/>

</video>
</Step>
</Steps>

### Remove linked feature flags

<Steps>
<Step>
Open any Item on your monday.com board with a linked feature flag.
</Step>
<Step>
Remove the linked feature flag by opening the **More menu** and selecting the **Unlink** option.
