# Project Overview

This is the ConfigCat documentation repository, a Docusaurus-based documentation site for ConfigCat's feature flag and configuration management service. The site is available at https://configcat.com/docs.

# General Instructions:

When reviewing pull requests:
- Only check .md and .mdx files that changed in the PR.
- Focus on text content and formatting.
- Prioritize actionable fixes.
- Add comments with code suggestions if possible.

# Review Focus

When reviewing documentation changes, check for:
- Typos and grammatical errors.
- Adherence to the writing style and text formatting guidelines defined by ConfigCat's Guidelines for Text located at @./GUIDELINES.md
- Accuracy of any code snippets or technical instructions.
